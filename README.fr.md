# 🇨🇭 Kit de Conformité nLPD pour Microsoft 365 Copilot

🇫🇷 Français | 🇬🇧 [English version](README.md)

🔐 Sécurisez votre environnement Microsoft 365 Copilot et atteignez la conformité nLPD en quelques heures — pas en semaines.

---

## 🎯 Qu’est-ce que c’est ?

Un kit de conformité pratique et prêt à l’emploi pour :

- Détecter les données sensibles suisses (AVS, IDE, IBAN, etc.)
- Configurer Microsoft Purview (DLP, audit, politiques)
- Réduire les risques liés à Microsoft 365 Copilot
- Se préparer aux contrôles du PFPDT

---

## ⚠️ Pourquoi c’est important

Avec la nouvelle loi suisse nLPD (RS 235.1), les organisations doivent :

- Savoir où se trouvent les données sensibles  
- Contrôler l’accès et l’exposition  
- Garantir la traçabilité et l’accountability  

➡️ Microsoft 365 Copilot introduit de nouveaux risques s’il est mal configuré
